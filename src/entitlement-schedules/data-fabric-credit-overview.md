1. Workflow Data Fabric
Workflow Data Fabric (WDF) is a unified framework from ServiceNow that empowers customers
to connect, understand, and act on data across their enterprise with real-time intelligence.

2. Data Fabric Credits

action or usage event‚Äî such as executing an RPA bot, making an API call, or transferring data
